## Adding Screenshots

When capturing screenshots:

1. **Resolution**: Use 1920x1080 for desktop
2. **Format**: PNG (lossless)
3. **Naming**: `[role]/[feature]-[variation].png`
4. **Optimize**: Compress images before committing
5. **Update**: Reference screenshots in main documentation
