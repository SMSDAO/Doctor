## Overview

AlgoBrainDoctor provides role-based workflows for different user types:

### Operator Role
- Monitor repository health scores
- Track worker status and performance
- Review Healdec auto-healing actions
- Manage watchlist and alerts

### Admin Role
- Control worker lifecycle (start/stop)
- Manage developer identities
- Configure system settings
- Monitor job queue

### Analyst Role
- View system-wide analytics
- Analyze health score distribution
- Review Healdec strategy effectiveness
- Generate performance reports

### Developer Role
- Manage API keys
- Test API endpoints in playground
- Review API documentation
- Integrate with external systems

## Common Tasks

### Monitor a Repository
1. Select Operator role
2. Find repository in list
3. Click star to add to watchlist
4. Set alert threshold if needed

### Check Worker Health
1. Navigate to Workers tab
2. View heartbeat indicators
3. Check success rates
4. Review recent job history

### Generate Reports
1. Select Analyst role
2. Choose report type
3. Set date range
4. Export results
